# Pending Restaurant Table Booking System

Welcome to the Pending Restaurant Table Booking System! This project allows users to book tables at the Pending restaurant. Users can register, login, and manage their table bookings conveniently.

## Technologies Used

- Frontend: React
- Backend: Spring (Java)
- Database: MySQL
